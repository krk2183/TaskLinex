# TaskLinex

TaskLinex is a web-based productivity tool designed to **automate and enhance workflow management**. It provides a clean interface for task organization, project tracking, and team collaboration.

---

## Features

- **Collapsible Navigation Sidebar**: Easily access different sections of your workspace.
- **Dark Mode Toggle**: Switch between light and dark themes for better accessibility.
- **Responsive Layout**: Works seamlessly on desktop and mobile.
- **Next.js + Tailwind CSS**: Built with modern web technologies for fast performance.

---
